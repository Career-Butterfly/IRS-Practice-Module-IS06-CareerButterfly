.NET Multi-platform App UI (MAUI) is a cross-platform framework that enables developers to create native mobile and desktop apps using a single C# codebase for Android, iOS, macOS, and Windows. It abstracts the latest technologies into a common framework built on .NET, allowing you to build apps that look and feel like native platforms1.

  4. Announcing the .NET MAUI Community Toolkit v1.3: Learn about the latest version and its features.
