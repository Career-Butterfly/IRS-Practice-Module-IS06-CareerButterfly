The AWS Cloud Development Kit (CDK) is an open-source software development framework that allows you to define cloud infrastructure in code and provision it through AWS CloudFormation. It consists of two primary parts: the AWS CDK Construct Library, which provides reusable code constructs for building infrastructure, and the AWS CDK Toolkit, a command-line tool for managing CDK projects12.
