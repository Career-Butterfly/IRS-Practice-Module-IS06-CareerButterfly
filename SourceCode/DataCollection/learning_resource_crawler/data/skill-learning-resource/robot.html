Robot Framework is a generic open-source automation framework used for acceptance testing, acceptance test-driven development (ATDD), and robotic process automation (RPA). It features a simple plain text syntax and can be extended with libraries implemented using Python or Java. Robot Framework is operating system and application independent12.
