A MAC address, short for Media Access Control Address, is a 48-bit hardware identification number that uniquely identifies each device on a network. It serves as a unique identifier for network interfaces, such as Ethernet cards or Wi-Fi adapters, and is used by routers and other network equipment to track connected devices1.
