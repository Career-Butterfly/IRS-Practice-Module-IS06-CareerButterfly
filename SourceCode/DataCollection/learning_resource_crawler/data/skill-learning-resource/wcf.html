Windows Communication Foundation (WCF), previously known as Indigo, is a free and open-source framework within the .NET Framework. It allows you to build connected, service-oriented applications by enabling asynchronous message communication between service endpoints. These endpoints can be part of a continuously available service hosted by IIS or an application1.
