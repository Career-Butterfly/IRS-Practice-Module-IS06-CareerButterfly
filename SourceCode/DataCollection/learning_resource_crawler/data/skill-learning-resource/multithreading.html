Multithreading in computer architecture refers to the ability of a central processing unit (CPU) or a single core in a multi-core processor to provide multiple threads of execution concurrently, supported by the operating system. Unlike multiprocessing, where multiple complete processing units are involved, multithreading aims to increase utilization of a single core by using thread-level parallelism and instruction-level parallelism12.
