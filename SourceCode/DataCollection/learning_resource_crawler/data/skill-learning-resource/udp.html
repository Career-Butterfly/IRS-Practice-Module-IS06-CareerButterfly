User Datagram Protocol (UDP) is a lightweight data transport protocol used primarily for low-latency and loss-tolerating connections between applications on the internet. Unlike TCP, UDP does not guarantee data delivery or retransmit lost or corrupted messages, making it suitable for real-time applications like voice over IP (VoIP), domain name system (DNS) lookup, and video or audio playback123.
