YOLO (You Only Look Once) is a state-of-the-art, real-time object detection algorithm introduced in 2015. Unlike traditional methods, YOLO processes entire images in a single pass, making it ideal for real-time applications. It spatially separates bounding boxes and associates probabilities to detected objects using a single convolutional neural network (CNN) 1.
