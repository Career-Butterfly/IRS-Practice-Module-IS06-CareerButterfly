.NET Multi-platform App UI (MAUI) is a cross-platform framework for creating native mobile and desktop apps using C# and XAML. It allows you to build apps that run on Android, iOS, iPadOS, macOS, and Windows from a single shared codebase123.

Here are five free resources to learn more about .NET MAUI:

  1. .NET MAUI for Beginners: This series covers the basics of building multi-platform apps with .NET MAUI for iOS, Android, macOS, and Windows using a shared C# codebase.
  2. GitHub Repository: Explore the official GitHub repository for .NET MAUI, which includes documentation, sample code, and community contributions.
  3. Microsoft Learn: Dive into tutorials, videos, and code samples on Microsoft Learn to learn how to use .NET MAUI effectively.
  4. .NET Podcast App: Check out the .NET Podcast app, a sample application showcasing .NET 6, ASP.NET Core, Blazor, .NET MAUI, Azure Container Apps, and more.
  5. Visual Studio Integration: Leverage the power of Visual Studio to build beautiful apps faster and easier with .NET MAUI.
