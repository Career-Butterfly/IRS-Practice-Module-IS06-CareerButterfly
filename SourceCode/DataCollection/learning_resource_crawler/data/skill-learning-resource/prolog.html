Prolog is a logic programming language that has its origins in artificial intelligence, automated theorem proving, and computational linguistics. Unlike many other programming languages, Prolog is primarily intended as a declarative programming language, where the program consists of a set of facts and rules defining relations. Computation in Prolog is initiated by running queries over the program1.
