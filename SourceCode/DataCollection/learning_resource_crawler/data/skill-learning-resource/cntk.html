The Microsoft Cognitive Toolkit (CNTK) is an open-source toolkit designed for commercial-grade distributed deep learning. It represents neural networks as a series of computational steps using a directed graph, allowing users to easily create and combine popular model types such as feed-forward DNNs, convolutional neural networks (CNNs), and recurrent neural networks (RNNs/LSTMs). CNTK supports stochastic gradient descent (SGD) with automatic differentiation and parallelization across multiple GPUs and servers. Additionally, it is one of the first deep-learning toolkits to support the Open Neural Network Exchange (ONNX) format, enabling seamless model transfer between frameworks like CNTK, Caffe2, MXNet, and PyTorch1.
