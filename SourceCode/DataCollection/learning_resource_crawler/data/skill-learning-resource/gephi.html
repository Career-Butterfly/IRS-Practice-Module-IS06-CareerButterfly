Gephi is an open-source network analysis and visualization software written in Java on the NetBeans platform. It is primarily used for visualizing, manipulating, and exploring networks and graphs from raw edge and node graph data. With Gephi, you can explore, analyze, spatialize, filter, cluster, manipulate, and export various types of graphs123.
