AWS Device Farm is an application testing service that allows you to improve the quality of your web and mobile apps by testing them across an extensive range of desktop browsers and real mobile devices hosted in the AWS Cloud. It eliminates the need to provision and manage testing infrastructure, enabling you to run tests concurrently on multiple devices and generate videos and logs for quick issue identification1.
