Amazon Simple Notification Service (Amazon SNS) is a managed messaging service that facilitates communication between publishers and subscribers by allowing asynchronous message delivery through topics. Subscribers can receive messages via various endpoints, including Amazon Data Firehose, Amazon SQS, AWS Lambda, HTTP, email, mobile push notifications, and SMS1.
