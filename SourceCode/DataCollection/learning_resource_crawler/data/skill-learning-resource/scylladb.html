ScyllaDB is an open-source distributed NoSQL wide-column data store designed to be compatible with Apache Cassandra while achieving significantly higher throughputs and lower latencies1. It stores data using topographical data models, where each entity is represented as a node, and relationships between nodes are represented by edges2.
